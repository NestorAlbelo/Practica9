    <h1 class="class">class Fracciones</h1>

<div id="description" class="description">

    <h1 id=“label-Clase+Fracciones.”>Clase <a href=“Fracciones.html”>Fracciones</a>.<span><a href=“#label-Clase+Fracciones.”>&para;</a> <a href=“#documentation”>&uarr;</a></span></h1>

    <p>La finalidad de esta clase es la de representar números racionales de la forma numerador/denominador y comprender un conjunto de operaciones. Estas operaciones son:</p> <ul><li> <p>to_s.</p> </li><li> <p>flotante: pasa el numero de la forma numerador/denominador a número flotante.</p> </li><li> <p>abs : valor absoluto.</p> </li><li> <p>reciproco: numerador/denominador -&gt; denominador/numerador.</p> </li><li> <p>opuesto: numerador/denominador -&gt; -numerador/denominador.</p> </li><li> <p>Suma.</p> </li><li> <p>Resta.</p> </li><li> <p>Multiplicar.</p> </li><li> <p>División.</p> </li><li> <p>Módulo.</p> </li><li> <p>Coerce.</p> </li><li> <p>&lt;=&gt; para este operador es necesario incluir el módulo Comparable.</p> </li></ul>

    </div><!-- description -->
